# What a City Average Hides
## Global Air-Quality Fingerprints Using OpenAQ and Open-Meteo

This project was developed for the **02806 Social Data Analysis and Visualization** course at **DTU, Denmark**.

The project investigates how urban air pollution changes across cities, seasons, weather conditions, and particle profiles. Instead of ranking cities only by annual average PM2.5, the project asks:

> **What does a city average hide?**

Using OpenAQ air-quality data and Open-Meteo weather data, we build a coverage-aware city dataset and create an interactive magazine-style webpage that explains global PM2.5 patterns, daily exceedance burden, particle profiles, weather-linked episodes, and a Dubai-focused deep dive.
